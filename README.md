# ClockRhythm2
## 1. 개요

유니티 버전: Unity 2019.4.17f1

코드 또는 프로젝트 구조가 궁금하신 분들을 위해 올렸으며 추후에 삭제될 수 있습니다.

## 2. 참고사항

히트사운드는 저작권 우려상 뺐기 때문에 임의의 효과음을 구하셔서 직접 HitsoundPlayer 스크립트에 대입하셔야 합니다.

게임의 시작 씬은 MenuScene 입니다.
